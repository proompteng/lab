# First Town

Your first town is the engine of the first two weeks. It should produce units, unlock mechanics, and support your main hero's route.

## Opening questions

- Which resources block my next dwelling?
- Which mine can I safely claim this week?
- Which unit growth matters most for safe fights?
- Which upgrade changes a real fight instead of looking nice?
- Can the town survive if the main hero leaves?

## Beginner rule

Prioritize recurring value. Dwellings, recruitment, and resource flow usually matter before late luxury.
