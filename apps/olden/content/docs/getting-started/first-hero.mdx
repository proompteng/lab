# First Hero

Your first main hero should do three things: clear early fights safely, keep moving through useful rewards, and level into a coherent plan.

## Choosing the main hero

Prefer a hero whose starting army and skills support the faction's early plan. If the hero starts with a useful spell or preservation tool, use it to reduce losses instead of saving mana forever.

## Early habits

- Keep the best army with the main hero.
- Use scouts for information and safe collection.
- Avoid leveling into unrelated jobs.
- Revisit the town only when recruitment, defense, or routing makes it worth the movement.
