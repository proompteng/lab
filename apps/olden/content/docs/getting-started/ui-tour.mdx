# UI Tour

The interface should be read by decision area: map, town, hero, army, combat, and end-of-turn state.

## Adventure screen

Use the adventure screen to track movement, revealed objects, hero routes, enemy approach, and town defense. The most important question is whether the selected hero can still perform a useful action this turn.

## Town screen

Use the town screen to inspect buildings, recruitment, upgrades, defenses, spells, and faction mechanics. Check tomorrow's building cost before spending all gold today.

## Hero screen

Use the hero screen to inspect skills, army stacks, artifacts, spells, attributes, mana, focus-related abilities, and role fit.

## Combat screen

Use the combat screen to track initiative, retaliation, obstacles, range, spell value, and whether a unit can survive after acting.
