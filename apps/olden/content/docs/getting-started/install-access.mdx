# Install and Access

Olden Era is available in Early Access. The practical first step is to install the current client, launch a low-pressure mode, and keep patch freshness in mind while reading guides.

## First launch path

1. Install the game from the official store page.
2. Start with the campaign, a small classic map, or single hero mode.
3. Pick one faction and avoid changing factions every run.
4. Play the first seven turns, then review route and combat mistakes.

## Settings to decide before multiplayer

- Map or scenario.
- Timer rules.
- Restart policy.
- Whether players are learning, practicing, or competing.
- Whether Early Access bugs or disconnects trigger reloads.
