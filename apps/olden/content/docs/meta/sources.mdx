# Sources and Freshness

Olden Era is in Early Access, so facts can change. Every factual page should list source links and the version baseline used when it was written.

## Current baseline

- Last verified: 2026-05-31
- Client patch observed on official wiki: v0.80.16
- Release state: Early Access

## Source priority

1. Official game pages, Steam, publisher/developer announcements, and official wiki pages.
2. Direct in-game verification when available.
3. Original player strategy derived from repeatable gameplay reasoning.
4. Community commentary only when clearly labeled as opinion or meta, not fact.

## Refresh rules

- Refresh faction, unit, skill, spell, and law pages after balance patches.
- Refresh mode pages after roadmap items ship.
- Refresh strategy pages when a mechanic changes or a common route stops working.
- Do not copy long third-party guide prose. Rewrite strategy in original language and link sources for facts.
