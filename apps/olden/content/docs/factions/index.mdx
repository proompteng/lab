# Factions

Olden Era currently has six Early Access factions. The best first faction is not automatically the strongest faction; it is the faction whose decisions you can read clearly enough to improve.

<FactionGrid />

## Choosing a first faction

- Choose Temple if you want clear fundamentals.
- Choose Necropolis if attrition and undead snowballing sound intuitive.
- Choose Grove if movement, nature, and map-flow planning are appealing.
- Choose Dungeon if you like tactical attacks and high-ceiling combat.
- Choose Hive if you want synergy pressure and swarm commitment.
- Choose Schism if denial, summons, and control sequencing are the draw.
