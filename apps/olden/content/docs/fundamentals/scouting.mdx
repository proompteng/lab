# Scouting Fundamentals

Scouting is how you avoid committing the main hero blindly. It reveals which route has mines, which fight gates a dwelling, and where opponents can pressure.

## What to scout

- Road branches.
- Mine guards.
- Town approaches.
- Bank and artifact guards.
- Enemy hero direction.
- Safe loose resources.

## Beginner rule

Scout before the main hero reaches the branch, not after.
