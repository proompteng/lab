# Morale, Luck, and Focus

Morale, luck, and focus add volatility and ability pacing to combat. Strong players reduce bad volatility for themselves and force awkward volatility on opponents.

## Morale

Morale can create extra action value. Armies that rely on positive morale should protect the stacks most likely to convert an extra action into a kill, block, or saved unit.

## Luck

Luck changes damage spikes. It is strongest when a lucky hit lands on a stack whose death changes the round. It is weakest when it overkills a low-value target.

## Focus

Focus powers important hero and creature abilities. Treat focus like mana with a shorter combat loop: spend it when the ability changes the fight, not merely when the button lights up.

## Beginner rule

Plan as if the normal result happens, then use morale, luck, and focus to improve a good plan. Do not rely on lucky rolls to rescue a bad route.
