# Movement

Movement is the hidden currency. A hero with army but no movement cannot claim the mine, reach the town, or start the fight that matters.

## Beginner rule

End turns near the next useful action. If a hero stops in empty space, the route probably needs correction.

## Practical movement habits

- Prefer roads when the route value is close.
- Reveal fog before committing to a long dead-end.
- Let scouts collect safe piles so the main hero keeps fighting.
- Avoid sending the main army back to town unless recruitment, defense, or teleport/movement tools justify it.
- Count whether the hero can fight and still reach the next object.

## Advanced note

Movement advantage compounds when it creates first access to mines, banks, dwellings, and towns. It is often correct to skip a medium fight if doing so reaches a strategic object one turn earlier.
