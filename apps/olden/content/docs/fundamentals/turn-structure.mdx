# Turn Structure

Good turns follow a loop: inspect, route, fight, build, recruit, scout, and end near the next useful action.

## Turn loop

1. Check town build and recruitment options.
2. Inspect hero movement before moving.
3. Choose the route with the best future payoff.
4. Fight only when the reward justifies expected losses.
5. Use scouts before committing the main hero.
6. End with tomorrow's first action already planned.

## Common mistake

Moving heroes first and checking town after. That often wastes movement because the building you choose changes which resource or fight matters.
