# Focus and Abilities

Focus and creature abilities make combat less about raw attacks and more about timing. The right ability on the right round can save army or create a decisive kill.

## Beginner rule

Spend focus when it changes the trade. Holding focus forever wastes it, but spending it on a harmless target is just as bad.

## Ability timing

- Use defensive abilities before the damage lands.
- Use control abilities before the enemy's key action.
- Use burst abilities when the target dies or becomes safe to finish.
- Use movement abilities to create a second threat, not only to move farther.
