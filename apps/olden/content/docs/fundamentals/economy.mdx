# Economy

Economy is not separate from combat. It decides how many creatures you can recruit, which buildings arrive on time, and whether a risky fight is needed for a bottleneck resource.

## Beginner rule

Secure wood, ore, gold, and the rare resource your town needs before buying luxury upgrades. A building that produces weekly growth is usually more important than an upgrade that helps only today's army.

## Resource priorities

- Gold recruits units, buys heroes, and keeps recovery possible.
- Wood and ore unlock the basic town plan.
- Rare resources decide faction-specific dwellings, magic, and late power.
- Mines are long-term income; loose piles are timing fixes.

## Build order logic

Build toward what changes the next week. If a dwelling creates growth every week, delaying it often costs more than the listed price. If an upgrade saves a fragile stack in the next fight, it can beat a slower economy build.

## Common mistake

Spending all gold on recruits before checking tomorrow's building requirement. The result is a strong-looking army that misses its next growth step.
