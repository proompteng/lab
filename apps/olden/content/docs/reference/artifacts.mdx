# Artifacts Reference

Artifacts should be assigned to the hero who can convert them into immediate value.

## Assignment rules

- Movement artifacts belong on the hero whose route matters this turn.
- Combat artifacts belong on the main hero before dangerous fights.
- Spell artifacts belong on heroes with mana and spell access to use them.
- Economy artifacts can sit on support heroes if they do not need to fight.

## Fight decision

Before taking an artifact fight, ask whether the item changes a future battle enough to repay the losses. If the guarded stack costs your main army too much, the artifact can be a losing reward.
