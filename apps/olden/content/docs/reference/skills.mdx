# Skills Reference

Skills are a commitment. A hero with scattered skills is rarely better than a hero with one clear job.

## Main hero filters

- Does this skill reduce losses?
- Does it make the main army faster or stronger?
- Does it unlock the spell tier or ability timing I need?
- Does it help the next three fights, not just the final fight?

## Scout filters

- Does this skill reveal more map?
- Does it save movement?
- Does it keep the scout alive without taking main-army troops?

## Economy filters

Economy skills are strongest when they accelerate a real build timing. If they pay back after the decisive fight, they were probably too slow.
