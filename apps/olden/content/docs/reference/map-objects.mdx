# Map Objects Reference

Map objects are the route puzzle. Each one should be judged by reward, guard cost, movement cost, and timing.

## Priority questions

- Does this object unlock a mine, dwelling, town, level, spell, or artifact?
- Is the guard fight cheap enough for the reward?
- Does taking it put the hero closer to the next objective?
- Can a scout safely collect it instead of the main hero?
- Will the opponent reach a contested object first if you delay?

## Object types

Mines and dwellings usually matter because they pay repeatedly. Banks and guarded rewards matter when the army can clear them without losing the next route. Shrines and stat objects matter when they support the hero's real job.
