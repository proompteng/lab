# Heroes Reference

Heroes should be evaluated by job, not only by raw strength. A strong main hero, useful scout, and support specialist solve different problems.

## What to check

- Specialty: what gets better as this hero levels?
- Starting army: can the hero take early fights safely?
- Starting skills: do they support fighting, scouting, economy, or magic?
- Starting spell: does it reduce losses or unlock a route?
- Faction fit: does the hero reinforce the town plan?

## Main hero standard

The main hero should make the best army safer and faster. Pick combat, magic, and movement options that help the next route.

## Scout standard

A scout should reveal information and collect safe resources without stealing the main army's timing.
