# Scenarios

Scenarios are authored maps with specific pacing, objectives, and constraints. The correct route is often map-specific.

## How to play

- Read win and loss conditions first.
- Identify whether the map rewards economy, rush pressure, defense, or exploration.
- Treat unusual terrain and object placement as intentional clues.
- Avoid applying a generic build order when the scenario clearly asks for a different plan.
