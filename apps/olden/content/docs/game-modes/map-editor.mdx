# Map Editor

The map editor is useful for creating practice maps, testing faction starts, and building authored scenarios.

## Good practice maps

- Small maps for first-week route drilling.
- Combat maps with one repeated neutral fight.
- Town-development maps that test resource bottlenecks.
- Multiplayer test maps with mirrored starts.

## Early Access note

Editor workflows are expected to improve during Early Access, so validate custom maps after patches.
