# Campaign

Campaign missions teach the setting, scripted objectives, and authored pressure. Treat them as puzzle maps with story constraints, not as pure sandbox tests.

## How to play

- Read objectives before optimizing a generic route.
- Preserve army because campaign recovery can be more constrained than skirmish recovery.
- Expect mission scripts to change priorities.
- Use campaign missions to learn UI, combat, and faction basics before competitive play.
