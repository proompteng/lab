# Arena

Arena mode isolates battles from the larger map economy. Use it to learn initiative, retaliation, spell timing, ability timing, and faction army roles.

## How to practice

- Replay the same matchup with different first moves.
- Track which stack decided the battle.
- Test whether a spell saves more army than direct damage.
- Learn counters without spending a full map to reach the fight.

## Limitation

Arena does not teach town timing, scouting, mines, or growth. Use it for combat, then return to classic maps for full strategy.
