# Classic Mode

Classic mode is the full Heroes-style loop: map exploration, several heroes, towns, mines, neutral armies, economy, and decisive battles.

## How to play

- Build around week timing.
- Use scouts without weakening the main hero.
- Claim mines and dwellings before chasing every small reward.
- Track opponent reach once contact is possible.
- Win by making the final battle unfair before it starts.
