# Single Hero Mode

Single hero mode removes much of the chain and scout complexity. The result is a cleaner test of route quality, hero development, and combat preservation.

## How to play

- Plan movement more carefully because there are fewer support heroes.
- Pick skills that support one coherent main-hero job.
- Avoid fights that leave the only hero unable to keep clearing.
- Use the mode to learn faction combat before full classic maps.
